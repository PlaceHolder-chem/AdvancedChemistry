# Логика и дизайн

## Ресурсы.
- Оксид Серы (SO2)
    - Получение: Атмосферная центрефуга (Напрямую)
    - Газ
    - Тег: sulfur-dioxide

    - Сера (S)
        - Получение: Восстановитель Серы (2SO2 + CH4 -> 2S + CO2 + H2O)
        - Твердый материал
        - Тег: sulfur

    - Сероводород (H2S)
        - Получение: Аппарат Киппа (FeS + 2HCl -> FeCl2 H2S)
        - Газ

        - Хлорид Железа
            - Получение: Аппарат Киппа
            - Твёрдый материал
            - основное применение: Снаряд
    
    - Серная кислота (H2SO4)
        - Получение: Пероксидная лаборатория (Оксид Серы (SO2) + Перекись Водорода (H2O2))
        - Жидкость
        

- Минерал
    - Получение: Минеральная руда (Буром)
    - Добыча: Пирит (FeS), NaCl (Соль), Al2O3 (Оксид алюминия), PbO2 (Оксид свинца (IV)), ZnCl (Хлорид Цинка), LiF (Фторид лития), Апатиты, Ag (СереБРО)

    - Пирит
        - Получение: Расщепитель минералов (Минерал)
        - Твёрдый ресурс

    - Соль
        - Получение: Расщепитель минералов (Минерал)
        - Твёрдый ресурс

    - Оксид Алюминия
        - Получение: Расщепитель минералов (Минерал)
        - Твёрдый ресурс

    - Оксид Свинца
        - Получение: Расщепитель минералов (Минерал)
        - Твёрдый ресурс

    - Хлорид Цинка
        - Получение: Расщепитель минералов (Минерал)
        - Твёрдый ресурс

    - Фторид Лития
        - Получение: Расщепитель минералов (Минерал)
        - Твёрдый ресурс

    - Апатиты
        - Получение: Расщепитель минералов (Минерал)
        - Твёрдый ресурс

    - Серебро
        - Получение: Расщепитель минералов (Минерал)
        - Твёрдый ресурс

- Метан (CH4)

    - Получение: Атмосферная центрефуга (Напрямую)
    - Газ
    
    - Углерод (C)
        - Получение: Пиролизатор Метана (CH4 t -> C + 2H2)
        - Твердый ресурс
    
        - Карбон
            - Получение: Вакуумный Инфузор (C + Акрицит, зачтём за Смолу)
            - Твёрдый ресурс

- Хлор (Cl2)
    - Получение: Электролизер
    - Газ

    - Соляная Кислота
        - Получение: Синтезатор Хлороводорода (H2 + Cl2 -> 2HCl. + Water)
        - Жидкость

- Фосфор (Белый) (P)
    - Получение: Электропечь (Апатиты + Песок + Кремний)
    - Твёрдый материал

- Перекись Водорода (H2O2)
    - Получение: Катализатор Ракетного Топлива (Ракетное Топливо + Серебро)
    - Жидкость

## Производственные блоки

1. Атмосферная центрефуга.
    - Назначение: Извлекает из атмосферы Эрекира оксид серы (SO₂) и метан (CH₄)
    - Размер: 3
    - Вход: Энергия
    - Выход: SO₂ (0.5) + CH₄ (0.5) | Оксид серы, Метан за цикл
    - Тип: GenericCrafter

2. Пиролизатор метана
    - Назначение: Разлагает метан на углерод и водород
    - Размер: 3
    - Вход: CH4 (Метан) + Энергия
    - Выход: C (Углерод) + H2 (Водород)
    - Тип:

3. Восстановитель серы
    - Назначение: Превращает оксид серы и метан в чистую серу
    - Размер: 3
    - Вход: SO2 (Оксид Серы) + CH4 (Метан) + Энергия
    - Выход: S (Сера), Вода, Углекислый газ (побочка)
    - Тип:

4. Вакуумный инфузор
    - Назначение: Создает карбон
    - Размер: 4
    - Вход: Углерод (C) + Акрицит + Энергия
    - Выход: Карбон
    - Тип: GenericCrafter

5. Расщепитель Минералов
    - Назначение: Перерабатывает минеральную руду в набор полезных компонентов
    - Размер: 6
    - Вход: Минеральная Руда, Вода, Энергия
    - Выход: Пирит (FeS), NaCl (Соль), Al2O3 (Оксид алюминия), PbO2 (Оксид свинца (IV)), ZnCl (Хлорид Цинка), LiF (Фторид лития), Апатиты, Ag (СереБРО)
    - Тип:

6. Фосфорная электропечь
    - Назначение: Производит белый фосфор из апатитов
    - Размер: 3
    - Вход: Апатиты + Песок + Кремний + Много энергии
    - Выход: Белый фосфор
    - Тип: GenericCrafter

7. Аппарат Киппа
    - Назначение: Производит сероводород из пирита и соляной кислоты
    - Размер: 2
    - Вход: Пирит (FeS) + Соляная Кислота (HCl)
    - Выход: Сероводород (H2S) + Хлорид железа (FeCl2)
    - Тип: GenericCrafter

8. Пероксидная лаборатория
    - Назначение: Производит серную кислоту из оксида серы и перекиси водорода
    - Размер: 3
    - Вход: SO2 + H2O2 
    - Выход: H2SO4
    - Тип: GenericCrafter

9. Синтезатор хлороводорода
    - Назначение: Производит соляную кислоту из водорода и хлора
    - Размер: 4
    - Вход: Водород (H₂) + Хлор (Cl₂) + Вода
    - Выход: Соляная кислота (HCl)
    - Тип: GenericCrafter

10. Катализатор ракетного топлива
    - Назначение: Производит перекись водорода из ракетного топлива
    - Размер: 3
    - Вход: Ракетное топливо + Серебро + Энергия
    - Выход: H2O2
    - Тип: GenericCrafter
