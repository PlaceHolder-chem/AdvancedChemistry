# Advanced Chemistry

[![Mod Version](https://img.shields.io/badge/version-1.0.1-orange)](https://github.com/PlaceHolder/advanced-chemistry)
[![Mindustry Version](https://img.shields.io/badge/Mindustry-157.4-blue)](https://github.com/Anuken/Mindustry)

**Advanced Chemistry** — мод для Mindustry, добавляющий новые химические технологии и производственные цепочки на планету **Эрекир**.

---

## 🇷🇺 Русский

### Описание
Данный мод добавляет новый контент, связанный с химией на планете Эрекир.  
Мод находится в активной разработке, а также в бета-версии, так что ожидайте **несбалансированность** мода и возможные ошибки.

Спрайты будут обновлены в ближайшем будущем.

### Установка
1. Скачай последнюю версию мода как ZIP-архив.
2. Помести архив в папку `mods` Mindustry.
3. Запусти игру и активируй мод в меню "Моды".
---

## 🇬🇧 English

### Description
This mod adds new chemistry-related content to the planet **Erekir**.  
The mod is in active development and is in a beta version, so expect **imbalance** and possible bugs.

Sprites will be updated in the near future.

### Installation
1. Download the latest version of the mod as a ZIP archive.
2. Place the archive in the `mods` folder of Mindustry.
3. Launch the game and activate the mod in the "Mods" menu.
